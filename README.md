# CST-Student-Kit

Desktop app for University Students.

Functions
1. Automatic updating/download of lecture notes
2. Ease of use communication via mail
3. Contacts List keeping
4. Educational Forum
5. Notes Uploading
6. Free sms to any Number
7. Important Schedule Reminder
